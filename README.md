# 🎮 Games ClaudeCode

一个纯前端（HTML/CSS/JS）轻量游戏合集，部署于 **Vercel**，支持云端存档与排行榜。

> 🐍 🧱 🟦 💣 🐦 — **6 款经典小游戏，即点即玩**

---

## 📸 游戏总览

| 游戏 | 图标 | 说明 |
|:----:|:----:|:------|
| **贪吃蛇** | 🐍 | 经典贪吃蛇，键盘/触屏操控，15级速度渐进 |
| **俄罗斯方块** | 🧱 | 标准 10×20 棋盘，7-bag 随机，幽灵方块，墙踢 |

| **扫雷** | 💣 | 3种难度（初/中/高），首次安全，右键标旗，计时 |
| **Flappy Bird** | 🐦 | 重力物理飞行，管道碰撞，键盘/点击/触屏操控 |

## 🚀 快速部署

### 方式一：GitHub + Vercel（推荐）

```bash
# 1. 创建 GitHub 仓库并推送
git init
git add .
git commit -m "🎮 Games ClaudeCode"
gh repo create <仓库名> --public --push

# 2. 浏览器打开 https://vercel.com/new
#    导入该仓库 → 点击 Deploy → ✅ 完成
```

### 方式二：Vercel CLI

```bash
npm i -g vercel
vercel
```

---

## ☁️ 启用云端存档

部署后，前往 **Vercel Dashboard** 开启 Blob 存储：

1. 进入项目 → **Storage** → **Create → Blob Store**
2. 创建后复制 `BLOB_READ_WRITE_TOKEN`
3. **Settings → Environment Variables** → 添加：

| Name | Value |
|------|-------|
| `BLOB_READ_WRITE_TOKEN` | 上一步复制的 Token |

4. 回到 **Deployments** → 找到最新部署 → 点 **Redeploy**

之后所有用户分数自动同步到云端，排行榜功能同时生效。  
*未配置 Token 时自动降级为 `localStorage` 本地存储，功能不受影响。*

## 🎮 各游戏功能

### 🐍 贪吃蛇
| 功能 | 说明 |
|------|------|
| **操控** | `↑ ↓ ← →` / `W A S D` + 触屏按钮 + 滑动手势 |
| **输入队列** | 方向键 FIFO 队列，快速连按不丢失 |
| **速度系统** | 每吃 3 个食物升一级，共 15 级，速度条实时反馈 |
| **暂停** | `Space` 暂停/继续 |

### 🧱 俄罗斯方块
| 功能 | 说明 |
|------|------|
| **7 种方块** | I/O/T/S/Z/J/L，各自独立颜色和 3D 渲染 |
| **旋转墙踢** | 6 种偏移尝试，防止旋转卡墙 |
| **幽灵方块** | 半透明落点预览 |
| **7-bag 随机** | 标准 Tetris 公平随机算法 |
| **等级加速** | 每消 10 行升级，速度 800ms → 50ms |
| **计分** | 单行 100×Lv / 双行 300×Lv / 三行 500×Lv / 四行 800×Lv |

### 🏓 打砖块
| 功能 | 说明 |
|------|------|
| **物理** | 角度反射（击 paddle 位置决定反弹角度），速度递增 |
| **砖块** | 6 行 × 8 列，每行不同颜色/分值，前 3 行 2 条命 |
| **操控** | 鼠标追踪 + 键盘 `← →` / `A D` + 触屏按钮 |
| **生命** | 3 条命（❤️ 显示），全部失去则游戏结束 |

### 🟦 2048
| 功能 | 说明 |
|------|------|
| **操控** | 方向键 / WASD / 滑动手势 / 触屏按钮 |
| **动画** | 新方块 pop 弹出 + 合并 pulse 脉冲 |
| **计分** | 每次合并累加，最高分云端存储 |
| **胜利** | 合成 2048 后可继续挑战 |

### 💣 扫雷
| 功能 | 说明 |
|------|------|
| **难度** | 初级 9×9 (10雷) / 中级 16×16 (40雷) / 高级 30×16 (99雷) |
| **首次安全** | 首次点击及其相邻格绝不生成雷 |
| **标旗** | 右键标旗 🚩，长按触屏标旗 |
| **自动展开** | 点击空单元格自动展开相邻区域 |
| **计时** | 首次点击开始计时 |

### 🐦 Flappy Bird
| 功能 | 说明 |
|------|------|
| **物理** | 重力加速度 + 点击跳跃，飞行角度随速度变化 |
| **管道** | 固定间距 120px，随机高度，绿色渐变渲染 |
| **操控** | 鼠标点击 / 触屏 / `Space` / `↑` / `W` |
| **计分** | 通过管道得 1 分，高分本地+云端双存 |

## ☁️ API 接口

| 方法 | 路径 | 说明 |
|------|------|------|
| `POST` | `/api/score` | 提交分数 `{ game, score, player }` |
| `GET` | `/api/score?game=snake` | 获取某游戏排行榜 |
| `GET` | `/api/leaderboard` | 全局跨游戏排行榜 |

所有游戏共用一个 API，通过 `game` 参数区分。

## 🛠 技术栈

- **前端**: 原生 HTML5 + CSS3 + JavaScript (Canvas / DOM)
- **视觉**: 暗紫渐变背景 + 玻璃态毛玻璃效果

- **部署**: Vercel (Serverless Functions + Static Hosting)
- **CI/CD**: GitHub + Vercel 自动部署
